{{ title }}

Install Pickles with a package manager

{{#code}} yarn add https://bitbucket.org/relishinc/pickles.git {{/code}}

Then import the modules you need like so...

{{#code lang='js'}} import { Modal, FocusTrap } from 'pickles'; //=== console.log('Modal', Modal); console.log('FocusTrap', FocusTrap); // === let ft = new FocusTrap('#foo'); ft.stop(); {{/code}}
