ES6 module imports

Install Pickles with a package manager

yarn add https://bitbucket.org/relishinc/pickles.git

Then import the modules you need like so...

import jQuery from 'jquery';
import {
    Modal,
    FocusTrap
} from 'pickles';

//=== 

console.log('Modal', Modal);
console.log('FocusTrap', FocusTrap);

// ===

jQuery(document).ready(() => {

    let ft = new FocusTrap('#foo');
    ft.start();

});
